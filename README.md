# Termo Flutter

Projeto acadêmico desenvolvido em Flutter com o objetivo de criar um jogo inspirado no Termo (Wordle).

## Objetivo

Desenvolver uma aplicação mobile com mecânicas semelhantes ao jogo Termo, aplicando conceitos de desenvolvimento com Flutter.

## Referência

Este projeto segue como base o tutorial oficial do Flutter:

https://docs.flutter.dev/learn/pathway/tutorial/create-an-app
